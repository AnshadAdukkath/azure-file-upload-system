# Azure Cloud File Upload System

A cloud-based file upload application built using Node.js, Express.js, Multer, and Azure Blob Storage.

## Features
- Upload files to Azure Blob Storage
- Secure cloud storage
- Azure App Service deployment
- Environment variable configuration

## Technologies
- Node.js
- Express.js
- Multer
- Azure Blob Storage
- Azure App Service
